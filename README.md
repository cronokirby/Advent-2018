# AOC 2018

This repository contains some solutions for [advent of code 2018](http://adventofcode.com/2018)

## Running the solutions

Most of the solutions are in haskell, but here's instructions for running
everything:

### Haskell
```
runghc N.hs
```

### Python
```
python N.py
```

### Rust
```
rustc N.rs && ./N
```

### C
```
gcc N.c && ./N
```
